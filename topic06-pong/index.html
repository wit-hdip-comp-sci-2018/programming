In this session we will delve deeper into the Pong game and look at the Player class and its use of Arrays. We will also look at simple data analysis algorithms along with other techniques for outputting data.

labs
Lab-11c

On completion of this lab you should be familiar with the game of pong and have started building it. In this lab, you will work on PongGameV1_0 and PongGameV2_0.

Lab-12a

On completion of this lab you should be very familiar with the game of pong and have included collision detection code for the Ball and Paddle class. You should see the game evolve through maintaining tournaments, lives lost and scores for the game. In this lab, you will work on PongGameV3_0, PongGameV4_0 and PongGameV5_0.

Lab-12b

On completion of this lab you will have introduced an array of Players and build simple data analysis algorithms based on the stored data. In this lab, you will work on PongGameV5_0, PongGame7_0 and PongGameV8_0.

Lab-12c

On completion of this lab you should be familiar with the pythagorean theorem for collision detection. In this lab, you will work on PongGameV9_0.
